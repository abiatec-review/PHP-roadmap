
1 - Сделать файлохранилище с регистрацией и восстановлением пароля
2 - Должно быть 2 типа юзеров: админы и простые юзеры
3 - У простых юзеров должен быть CRUD только на свои файлы, у админов должен быть CRUD на всех юзеров и их файлы

В качестве фронтовой части можно юзать HTML-формы, или просто API
